# Google — Material Design 3 (Material You)

m3.material.io

## 4つの核心原則

- Material is metaphor — 物理世界から着想
- Bold & graphic — 大胆な色とタイポ
- Motion — 動きで意味を伝える
- Adaptive — デバイスに柔軟対応

## M3 の目玉機能

Dynamic Color — ユーザーの壁紙に合わせて自動カラー生成
柔軟なコンポーネント + アクセシビリティ重視

## 特徴 / 雰囲気

- 「実践的」「カスタマイズ自由」。Android / Web / ChromeOS を統一。
- エレベーション（影）で階層を表現。
- コンポーネントが豊富で日本語資料も多い。
- Figma 公式キットあり（無料）。
